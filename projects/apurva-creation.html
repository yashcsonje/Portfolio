About Project

Apurva Creation – Creative Agency Platform

Apurva Creation is a production-ready static website developed for a creative agency to showcase photography services, portfolios, and business information through a modern and responsive user interface. The application was built using Next.js, React.js, TypeScript, and Tailwind CSS, providing a fast, SEO-friendly experience across all devices.

The website is hosted on Amazon S3 and delivered globally through Amazon CloudFront, with secure HTTPS access enabled using AWS Certificate Manager (ACM). A complete CI/CD pipeline was implemented with GitHub Actions, enabling automated build and deployment whenever changes are pushed to the GitHub repository.

The project demonstrates practical AWS cloud hosting, static website deployment, secure cloud infrastructure, CDN-based content delivery, and DevOps automation—resulting in a scalable, secure, production-ready web application.

Key Highlights

  • Production-ready static website with Next.js, React.js, TypeScript, and Tailwind CSS
  • Amazon S3 hosting with global content delivery through CloudFront
  • Private S3 buckets protected by CloudFront Origin Access Control (OAC)
  • HTTPS/TLS security using IAM and AWS Certificate Manager (ACM)
  • Automated build and deployment through GitHub Actions CI/CD
  • Optimized performance with CloudFront caching and static asset delivery
  • Responsive, scalable, and secure architecture following AWS best practices
